Fish Movement Visualizations

By Jeffrey D Walker, PhD and Ben Letcher, PhD

A collection of data visualizations for exploring the movement of fish.

Source code available at https://github.com/walkerjeffd/sheds-fish-move

Crossfilter

  • crossfilter.html: Initial crossfilter using dc.js with bar charts for River, Year, Species, Section, Season, and Length.
  • crossfilter-scatter.html: crossfilter.html with a brushable scatter plot of weight vs length (poor performance, too many points?)
  • crossfilter-stack-river.html: crossfilter.html with river bar chart stacked by species
  • crossfilter-map.html: crossfilter.html with leaflet map and d3 overlay showing number of fish per section

Force Layout

  • force-mainstem-01.html: Force directed graph of fictious data moving between positions along a single crooked path
  • force-trib-01.html: Force directed graph of fictious data moving between positions along a linear path with a tributary branch

Parallel Coordinates

  • parallel-coord-time.html: Parallel coordinates of each individual fish moving between segments (y-axis) between seasons (y-axis) and colored by species
  • parallel-coord-season.html: Parallel coordinates of each individual fish moving between segments (y-axis) over time (x-axis) and colored by species

Maps

  • map-leaflet.html: Map of sections for each river using leaflet
  • map-leaflet-d3.html: Map of sections for each river using leaflet with D3 overlay
